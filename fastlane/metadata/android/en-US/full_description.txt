Simple launcher consisting of a scrollable list of apps with <a href="https://en.wikipedia.org/wiki/Predictive_text">predictive text</a> filtering using 4 buttons.

- short press launches app
- long press opens app settings
- back button removes last input
- home button clears input
- long press back button reloads list

<em>Example for input <code>[A_F][G_L][M_R][S_Z]</code></em>
